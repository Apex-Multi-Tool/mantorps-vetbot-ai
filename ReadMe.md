# Project Title

## Description
Mantorps-VetBot-Ai V0.1.2 BETA

Ai powered customer service Chat-Bot for a vet clinic
" Created for Mantorps SmÃ¥djursklinik AB "
 
LLM powered by Ollama and the Gemma3.14b modell.
Backend is a sqllite database for secure and fast handeling of customer data.
Ngnix V1.29 used as web server and loadbalancer
Frontend web ui is made with React V17.0 and React Native is used for the standalone Android app.

## Functions
* integrated customer self booking and apointment handeling service with google calander.

* Webhocks to Vet customer system

* Trained to handle function calling when needed and web search capable when asked to give general advice about pets. 

* Voice transcript handeling so customers can record messages by voice

* Push notifications when the vet sends reply or change bookings

* Integrated callcenter services to call back customers 

* Priority handeling cases

## Usage
Chat away! Ask about avalible times,prices or contact information but also for general information about vets or animals, anything you need to know about you sick pet.

The LLM is trained on the vet clinic it was made for first of all, But it is also trained on ALL webscraped content it could find about  veterinary clinics and sick animals.
